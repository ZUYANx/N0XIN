# NX0INFO

NX0INFO is a lightweight Python-based CLI tool designed to retrieve **name information associated with phone numbers** using an online lookup service.  
It focuses on simplicity, basic security validation, and a clean terminal interface.

## Features

- Phone number to name lookup  
- Automatic internet connectivity check  
- SSL certificate fingerprint verification  
- Clean and colorful CLI interface using `rich`  
- Optimized for Bangladeshi phone numbers  
- Lightweight and fast execution  
- Works on Termux and Linux environments
